Available API Endpoints

  Projects:
  - POST /api/projects - Create project
  - GET /api/projects - List all projects
  - GET /api/projects/{id} - Get project details
  - PATCH /api/projects/{id} - Update project
  - DELETE /api/projects/{id} - Delete project
  - POST /api/projects/{id}/files - Upload SAS files
  - GET /api/projects/{id}/workflow - Get workflow steps
  - PATCH /api/projects/{id}/workflow/{step_id} - Update workflow step

  Tasks:
  - GET /api/projects/{id}/tasks - List tasks
  - GET /api/projects/{id}/tasks/{task_id} - Get task details
  - PATCH /api/projects/{id}/tasks/{task_id} - Update task
  - POST /api/projects/{id}/tasks/{task_id}/regenerate - Regenerate conversion

  Comments:
  - GET /api/tasks/{task_id}/comments - Get comments
  - POST /api/tasks/{task_id}/comments - Add comment
  - PATCH /api/comments/{id} - Mark comment resolved

  Dashboard:
  - GET /api/dashboard/stats - Get statistics

  ---
  How It Works

  Crew Executes
      ↓
  Validate All Task Outputs
      ↓
  Any Failed? → Yes → translate_code failed?
      ↓                    ↓
      No                  Yes → Auto-fix (rename file)
      ↓                    ↓
  Success               Fixed? → Yes → Success
                          ↓
                         No → Retry Entire Crew (up to 2 times)
                          ↓
                      Still Failed? → Log Error, Mark Job Failed
